⚡ one-time setup needed
This board uses Firebase for real-time sync. It takes about 3 minutes:

1. Go to console.firebase.google.com → create a project (any name)
2. Click Realtime Database → Create database → Start in test mode
3. Go to Project Settings → Your apps → Web → Register app → copy the config
4. Replace the firebaseConfig object near the bottom of this file with your config
5. Save & re-upload — you're done! Share the link with friends 🎉

Until then, posts only show for you in this tab (no real-time sync).
